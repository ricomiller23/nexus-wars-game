# NEXUS WARS

A strategic dice-drafting board game with area control mechanics.

## Quick Start

Open `index.html` in a modern browser to play locally, or visit the live deployment.

## Game Overview

Nexus Wars is a two-player strategic board game featuring:
- **Dice Drafting**: Draft from a shared pool of 5 dice (3-2 split) each round
- **Dual Victory Conditions**: Control 5+ Nexus Points OR race your Champion to the opponent's Home Base
- **Circular Board**: 20 spaces with 7 special Nexus Points granting unique powers
- **Tactical Depth**: Blocking, bumping, and strategic positioning
